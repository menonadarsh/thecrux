01 What it does

⚿

Private by default

New repos are private — browsing and cloning are gated to the people you choose. Flip to public when you want.

⇅

Git over HTTP & SSH

Clone, fetch and push over Smart-HTTP or with your SSH keys. No daemons, no special client — just git.

◈

Revocable tokens

Push with per-device access tokens instead of your password. Shown once, stored hashed, revoked in a click.

◇

Issues & pull requests

Real diffs, labels and assignees, server-side merges (fast-forward or a true merge commit) with conflict detection.

◫

Organizations

Shared namespaces with owners and members. Group repos for a team — everyone in the org can push.

↯

Webhooks & audit log

Fire HMAC-signed post-receive payloads at your CI. Every security-relevant action lands in an append-only log.

⌘

Keyboard-first UI

A ⌘K command palette, vim-style navigation, light/dark themes. Server-rendered — works without JS.

⬡

One box, no database

A single Node app + Docker. Everything is files under one data dir, so a backup is a tar. No telemetry.
